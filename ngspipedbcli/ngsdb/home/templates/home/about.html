1. How to search? Top

You can use 'Search' to find feature which have name/alias of gene and gene annotation information that related to the search term entered.

2. How to use Blast? Top

BLAST service is based on the Django app django-blastplus (A simple Django app to conduct web-based blast+ local alignment search). For more details about BLAST programs and parameters, please visit https://www.ncbi.nlm.nih.gov/books/NBK279675/.

3. How to use genome browse? Top

Genome browser service is based on the IGV, which is a fast, interactive, genome visualization tool that's easy to embed in web pages and applications. For more details about Biodalliance, please visit here: https://github.com/igvteam/igv.js/.
